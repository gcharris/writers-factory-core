## How to Get Listed

Contributors are recognized in the following ways:

### Code Contributors
Submit a pull request to the repository. Once merged, you'll be added here with:
- Your name (or GitHub handle)
- Brief description of contribution
- Link to your project or website (optional)

### Course Students
Complete the "AI and the One-Week Novel" course. You'll be listed with:
- Your name
- Title of your novel/novella project
- Brief description of your work (optional)

### Documentation Contributors
Improve documentation, write tutorials, or create examples. You'll be listed with:
- Your name
- Type of documentation contributed
- Link to your work

### Bug Reporters & Feature Suggesters
Report bugs or suggest features that are implemented. You'll be acknowledged in:
- Release notes
- Commit messages
- This contributors file (for significant contributions)
