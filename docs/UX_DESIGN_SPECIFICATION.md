## 1. OVERVIEW & UX PHILOSOPHY

### Core Principles

**1. Invisible Complexity**
- Hide technical details (Cognee, API routing) - users just write
- Show only what matters: content, cost, progress
- Smart defaults that "just work"

**2. Keyboard-First Power**
- Every action accessible via keyboard
- Consistent shortcuts across screens
- Fast navigation between contexts

**3. Cost Consciousness**
- Always show session cost in status bar
- Warn before expensive operations
- Track and display cost breakdown on demand

**4. Seamless Flow**
- Auto-save everything (files, not database)
- Resume exactly where you left off
- No "save" button - system handles it

**5. Stage-Based Mental Model**
- Clear 5-stage pipeline: Creation â†’ Writing â†’ Enhancing â†’ Analyzing â†’ Scoring
- Current stage always visible
- Easy jumping between stages

### 11.2 Auto-Save System

```python
# Pseudocode for auto_save.py

class AutoSaver:
    def __init__(self, interval=30):
        self.interval = interval
        self.last_save = time.time()
        self.dirty = False

    async def mark_dirty(self):
        """Mark state as needing save"""
        self.dirty = True

    async def save_if_needed(self):
        """Check if save is needed and perform it"""
        if not self.dirty:
            return

        now = time.time()
        if now - self.last_save >= self.interval:
            await self.save()

    async def save(self):
        """Perform actual save"""
        # Save session state
        save_json("project/.session/current.json", {
            "current_stage": self.current_stage,
            "current_scene": self.current_scene,
            "model_preferences": self.model_preferences,
            "timestamp": time.time()
        })

        # Save cost tracking
        save_json("project/.session/costs.json", self.costs)

        self.dirty = False
        self.last_save = time.time()
```
